# TensorFlow JS SketchFace

![](./images/screenshot_1.jpg)


# Setup

```
cd ui
yarn install
```

# Running

In the `ui` folder:

```
python3 -m http.server 8003
```

Then, navigate to <http://localhost:8003/dist/index.html>.
